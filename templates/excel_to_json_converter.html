Excel to JSON Converter

Please note: Uploaded data will be deleted after conversion.

How To Use

  1. Download the EXCEL Template for Your Laboratory Test Data (Click the "Download Template" button below).
    2. Download Template
  2. Fill in the Excel Template with your Laboratory Test Data.
  3. Click on the "Choose File" button to select your Excel file.
  4. After selecting the file, click the "Convert" button.
  5. The converted JSON data will be displayed in your browser.
  6. You can click the "Copy JSON to Clipboard" button to copy the JSON data.
  7. You can now add the Laboratory Test Data as JSON to your project.
